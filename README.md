# Multiclass-Perceptron-from-Scratch
# [Project] Multiclass Perceptron Classifier from Scratch

본 프로젝트는 고려대학교 컴퓨터비젼(김창수 교수님) 수업의 과제로 진행되었으며, 외부 라이브러리(Scikit-learn 등)의 도움 없이 
**NumPy만을 활용하여 다중 클래스 퍼셉트론(Multiclass Perceptron) 분류기를 처음부터 끝까지(From-scratch) 구현**한 프로젝트입니다. 

## 🎯 프로젝트 개요
단층 퍼셉트론의 원리를 이해하고, 이를 확장하여 10개의 숫자 클래스(Digits dataset)를 분류하는 다중 클래스 분류기를 설계합니다. 단순히 결과만 도출하는 것이 아니라, 가중치 업데이트 규칙을 직접 코드로 구현함으로써 머신러닝의 기초 최적화 과정을 심도 있게 학습했습니다.

## 🛠 주요 특징 및 구현 내용
* **가중치 업데이트 로직 구현:** 틀린 예측에 대해 정답 클래스의 가중치는 높이고, 오답 클래스의 가중치는 낮추는 업데이트 규칙을 직접 구현했습니다.
* **모듈화된 설계:** 코드의 유지보수와 가독성을 위해 데이터 로드, 특징 추출, 학습, 테스트, 시각화 기능을 각 모듈(`.py`)로 분리하여 설계했습니다.
* **성능 평가 및 시각화:** 에폭(Epoch)에 따른 에러 변화를 모니터링하고, 테스트 데이터에 대한 예측 결과를 시각화하여 모델의 유효성을 검증했습니다.

## 📂 파일 구조 및 기능
* **`main.py`**: 전체 프로세스(학습 및 테스트)를 제어하는 메인 컨트롤러
* **`train.py`**: 퍼셉트론 학습 알고리즘 및 가중치 업데이트 핵심 로직
* **`features.py`**: 원시 이미지 데이터 전처리 및 특징 추출
* **`load_digits.py`**: Digits 데이터셋 로드 및 학습/테스트 데이터 분리
* **`test.py`**: 학습된 가중치를 활용한 예측 및 정확도 측정
* **`visualize_digit.py`**: 숫자 데이터 및 분류 결과 시각화 도구
